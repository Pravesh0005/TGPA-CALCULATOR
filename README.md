## 📊 How It Works

```
┌─────────────────────────────────────────────────────────────────┐
│                     TGPA CALCULATION ENGINE                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  You Enter:  Subject Name  |  Credits  |  Marks (out of 100)   │
│                                        ↓                        │
│  LPU Grade Scale:                                               │
│  91–100 → O   = 10.0  |  81–90 → A+  = 9.0                    │
│  71–80  → A   = 8.0   |  61–70 → B+  = 7.0                    │
│  51–60  → B   = 6.0   |  41–50 → C   = 5.0                    │
│  0–40   → F   = 0.0                                             │
│                                        ↓                        │
│  SGPA = Σ(Credits × Grade Points) / Σ(Credits)                 │
│                                        ↓                        │
│  TGPA = Σ(All Sem Credits × SGPA) / Σ(All Credits)            │
│                                        ↓                        │
│  Academic Health Score:                                         │
│  = min(100, round((TGPA × 8) + (avgMarks/10 × 2)))            │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 🎓 LPU Grade Table

| Marks | Grade | Grade Point | Result |
|-------|-------|-------------|--------|
| 91–100 | **O** | 10.0 | Outstanding |
| 81–90 | **A+** | 9.0 | Excellent |
| 71–80 | **A** | 8.0 | Very Good |
| 61–70 | **B+** | 7.0 | Good |
| 51–60 | **B** | 6.0 | Above Average |
| 41–50 | **C** | 5.0 | Average |
| 0–40 | **F** | 0.0 | Fail |

## 🌈 Design System

```css
/* Colour Palette — Dark Mode */
--bg-primary:    #050505    /* Near-black base */
--bg-glass:      rgba(255,255,255,0.04)   /* Glassmorphism */
--accent:        hsl(260, 80%, 65%)       /* Purple accent */
--success:       hsl(150, 60%, 45%)       /* Green for high TGPA */
--danger:        hsl(0, 70%, 55%)         /* Red for low/fail */

/* Colour Palette — Light Mode */
--bg-primary:    #f5f4f1    /* Warm off-white */
--bg-glass:      rgba(255,255,255,0.6)    /* Brighter glass */

/* Typography */
--font-body:     'Sora', system-ui
--font-mono:     'JetBrains Mono', monospace

/* Motion */
--spring:        cubic-bezier(0.16, 1, 0.3, 1)   /* Spring easing */
--stagger:       0.1s per element                  /* Entry stagger */
```

## 🛠 Tech Stack

```
Core:         Vanilla HTML5 + CSS3 + JavaScript (ES6+)
Animations:   CSS keyframes + Canvas API (particles + confetti)
Storage:      LocalStorage (auto-save, theme, semester data)
Export:       window.print() with print-specific CSS
Icons:        Inline SVG (Feather Icons style, zero external)
Fonts:        Google Fonts — Sora + JetBrains Mono
Framework:    None · Bundler: None · Dependencies: Zero
```

| Layer | Technology | Purpose |
|-------|-----------|---------|
| **Structure** | HTML5 | Semantic markup, single-file |
| **Styling** | CSS3 Custom Properties | Dual theme, glassmorphism, animations |
| **Logic** | Vanilla JS ES6+ | Grade calc, LocalStorage, events |
| **Particles** | Canvas API | Floating animated background |
| **Confetti** | Canvas API | Celebration on high TGPA |
| **Charts** | Pure CSS + JS | Grade distribution bars |
| **Export** | `window.print()` | Clean PDF report via browser |
